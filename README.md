# Execute in develop

1. Clone the repository
2. Ejecute

```
yarn
```

3. Nest CLI installed

```
npm i -g @nestjs/cli
```

4. Raise the database

```
docker-compose up -d
```

## Stack

- NestJS
- TypeORM
- MongoDB
- Docker
- Docker-compose
